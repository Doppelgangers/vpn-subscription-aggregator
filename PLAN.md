# План: Хаб подписок (Упрощенный)

## Суть
Приложение работает как прокси-агрегатор. Оно принимает запрос по UUID, опрашивает список внешних ссылок (подписок), объединяет результаты и отдает их клиенту.

## Модели
- `AggregateSubscription`: сущность с именем и UUID (токеном).
- `SourceLink`: список внешних URL, привязанных к агрегатору.

## Дизайн и Технологии (UI/UX)
- **Цветовая схема:** 
  - Primary: `#2563eb` (Blue 600)
  - Dark: `#1e293b` (Slate 800)
  - Accent: `#f59e0b` (Amber 500)
  - Background: `#f8fafc` (Slate 50)
- **Технологии:**
  - Frontend: Tailwind CSS (через CDN для простоты или локально).
  - Icons: Lucide Icons.
  - Components: Кастомные карточки и формы с единым border-radius (0.75rem).

## Функционал UI
- Кастомный дашборд для создания/редактирования агрегаторов.
- Никакой логики в Django Admin.
- AJAX/Fetch для добавления/удаления ссылок без перезагрузки (опционально).
